AI Built This

Welcome to a meta-documentation project: a website documenting its own creation by an AI agent.

The Concept

This entire website was built using only an AI agent (Strawberry), with a human simply providing decisions and approvals. Every line of code, every design choice, and every deployment step is documented.

Tech Stack

  • Static Site Generator: Eleventy (11ty)
  • Version Control: GitHub
  • Hosting: Stormkit
  • Design: Minimal black and white aesthetic
  • Domain: aibuiltthis.site

Why This Matters

As AI tools become more capable, understanding what they can build — and how — becomes crucial. This project is a transparent, step-by-step demonstration of what's possible when an AI agent has access to development tools.
